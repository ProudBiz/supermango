# Tasks: Delete a Todo

## Task 1: Add delete functionality
**Description:** Add a delete button to each todo item. Clicking it removes the todo from the list immediately.
**Acceptance Criteria:**
- Each todo item has a visible delete button
- Clicking delete removes that specific todo from the list
- Other todos remain unaffected
- The list re-renders correctly after deletion
**TDD Approach:** Test that clicking delete removes the specific todo from the rendered list; test that other todos remain after deletion
**Validation:** Run tests. Manually verify that deleting works correctly with multiple todos in the list.
