# Tasks: Dark Theme UI

## Task 1: Apply dark theme styling
**Description:** Style the entire app with a sleek, modern dark theme. Dark backgrounds, light text, subtle element contrast, clean typography, and responsive layout.
**Acceptance Criteria:**
- Dark background (#0a0a0a or similar) with light text (#e0e0e0 or similar)
- Input field, buttons, and list items have subtle contrast against the background
- Clean, modern font and generous spacing
- Layout is centered and readable on both mobile and desktop
- Hover states on interactive elements (buttons, checkboxes)
- Overall aesthetic feels polished and intentional, not just "dark background with white text"
**TDD Approach:** Test that the page renders with the expected dark theme container class applied
**Validation:** Run tests. Visually inspect in browser at various viewport widths (mobile, tablet, desktop) to confirm the design looks polished.
