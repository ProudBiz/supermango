# Tasks: Complete a Todo

## Task 1: Add toggle completion functionality
**Description:** Add a checkbox or toggle to each todo item. Clicking it toggles the todo's completed state. Completed todos should be visually distinct — strikethrough text and dimmed opacity.
**Acceptance Criteria:**
- Each todo item has a clickable checkbox
- Clicking the checkbox toggles the todo between active and completed
- Completed todos display with strikethrough text and reduced opacity
- Toggling back to active restores normal appearance
**TDD Approach:** Test that clicking the checkbox toggles the completed state; test that the completed CSS class is applied/removed correctly
**Validation:** Run tests. Manually verify toggling works and the visual distinction is clear.
