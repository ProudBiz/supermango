# Slack 링크 자동 요약

## Description
팀원이 Slack 채널에 링크를 포함한 메시지를 보내면, 봇이 자동으로 링크 내용을 3줄로 요약하여 스레드에 게시한다. 처리 상태를 이모지 리액션으로 표시한다.

## Acceptance Criteria
- 봇이 초대된 public/private 채널에서 링크 포함 메시지를 감지한다
- 링크 감지 시 원본 메시지에 👀 리액션을 추가한다
- Jina Reader로 URL 콘텐츠를 마크다운으로 추출한다
- Gemini API로 3줄 요약과 태그를 생성한다
- 스레드에 불릿 형태의 3줄 요약을 게시한다
- 요약 완료 시 👀을 제거하고 ✅ 리액션을 추가한다
- 한 메시지에 여러 링크가 있으면 하나의 스레드 메시지로 합산하여 요약한다
- URL이 5개 초과 시 처음 5개만 요약하고 나머지는 "요약되지 않은 링크 N개" 안내한다
- 요약 실패 시 ❌ 리액션과 실패 사유를 스레드에 안내한다
- DM에서는 동작하지 않는다
- 요약 결과를 Supabase DB에 저장한다
