# Tasks: 웹앱 랜딩페이지

## Task 1: 랜딩페이지 구현
**Description:** 서비스 소개와 Slack 로그인 CTA가 포함된 랜딩페이지를 구현한다. 디자인 톤은 Ralphthon 참고 — 딥 네이비 다크 배경, 대담한 타이포그래피, 노란색 액센트. 모바일 퍼스트 반응형.
**Acceptance Criteria:**
- 서비스 이름, 핵심 가치 소개가 표시된다
- "Slack으로 로그인" 버튼이 눈에 띄게 배치된다
- 클릭 시 Slack OAuth 플로우가 시작된다
- 딥 네이비 다크 배경 + 노란색 액센트 CTA
- 모바일/태블릿/데스크탑 반응형
**TDD Approach:** 페이지 렌더링 및 로그인 버튼 동작 테스트
**Validation:** gstack으로 모바일/데스크탑 뷰포트에서 랜딩페이지 확인
