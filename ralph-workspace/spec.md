# LinkDigest

## Overview
Slack 워크스페이스에서 공유되는 링크를 자동으로 요약하고 카드뉴스로 변환하는 SaaS. Slack Bot이 링크를 감지하면 3줄 요약과 카드뉴스(3장)를 생성하여 스레드에 게시하고, 별도 모바일 퍼스트 웹앱에서 모든 워크스페이스의 링크를 통합 피드로 모아볼 수 있다.

## Design Direction
**비주얼 톤:** 딥 네이비 다크 배경, 대담한 타이포그래피, 노란색 액센트, 터미널/모노스페이스 포인트 (Ralphthon 참고).

**웹앱:** 인스타그램 스타일 단일 피드 페이지. 카드뉴스 캐러셀 상단 + 공유자/날짜/요약/태그 하단. 무한스크롤. 상단에 워크스페이스/채널 드롭다운 + 태그 검색. 카드뉴스까지 완료된 메시지만 노출.

```
┌─────────────────────────┐
│ 🔗 LinkDigest    👤     │  ← 헤더
├─────────────────────────┤
│ [워크스페이스 ▼] [채널 ▼]│  ← 드롭다운 필터
│ [🔍 태그 검색...]       │  ← 태그 검색 입력
├─────────────────────────┤
│ ┌─────────────────────┐ │
│ │   카드뉴스 캐러셀    │ │
│ │   (1/3) ● ○ ○      │ │
│ └─────────────────────┘ │
│ 김철수 · 3시간 전       │
│ • 요약 1줄              │
│ • 요약 2줄              │
│ • 요약 3줄              │
│ [#개발] [#AI]    [공유] │
├─────────────────────────┤
│         ∞ 무한스크롤     │
└─────────────────────────┘
```

**Slack Bot:** 불릿 3줄 요약 스레드 메시지 → 카드뉴스 이미지 스레드 메시지. 리액션 흐름: 👀→✅→🖼️.

**공유 페이지:** 태그 + 캐러셀 + 요약 + CTA(랜딩페이지). OG: 3줄 요약 + 카드뉴스 1번 이미지.

```
┌─────────────────────────┐
│ 🔗 LinkDigest           │  ← 헤더 (로고만)
├─────────────────────────┤
│ ┌─────────────────────┐ │
│ │   카드뉴스 캐러셀    │ │
│ │   (1/3) ● ○ ○      │ │
│ └─────────────────────┘ │
│ • 요약 1줄              │
│ • 요약 2줄              │
│ • 요약 3줄              │
│ [#개발] [#AI]           │
├─────────────────────────┤
│ ┌─────────────────────┐ │
│ │  나도 써보기 →       │ │  ← CTA 버튼
│ └─────────────────────┘ │
└─────────────────────────┘
```

## Non-Goals
- 사용자가 직접 태그를 수정/추가하는 기능
- 링크 내용 전문 저장 (요약만)
- 모바일 네이티브 앱
- 결제/과금 시스템
- Slack 외 다른 메신저 지원
- 메시지당 6개 이상 URL 요약
- URL 기반 요약 캐싱/재사용

## Technical Considerations
- **모노레포:** Turborepo (`apps/web`, `apps/slack-bot`, `packages/shared`)
- **웹앱:** Next.js (App Router), Railway 배포
- **Slack Bot:** Bolt for JavaScript (Socket Mode), Railway 배포
- **DB/Auth/Storage:** Supabase (PostgreSQL + Slack OAuth OIDC + card-images 버킷)
- **AI:** Gemini API (3줄 요약 + 태그 생성)
- **스크래핑:** Jina Reader (URL → 마크다운)
- **카드뉴스:** 이미지 3장 고정, HTML/CSS 템플릿을 서버에서 이미지로 렌더링

## Risks
- Slack API rate limit — v1 별도 처리 없음, 모니터링
- Gemini API 지연/실패 — 타임아웃 + ❌ 리액션 + 사유 안내
- 카드뉴스 이미지 생성 시간 — 요약과 비동기 분리
- 스크래핑 차단 — Jina Reader + 실패 사유 안내
- Slack OAuth 토큰 만료 — Supabase Auth 자동 refresh
