# 웹앱 메시지 공유

## Description
사용자가 피드에서 메시지 단위로 공개 공유 링크를 생성하고, 외부 사용자가 로그인 없이 해당 링크에서 요약과 카드뉴스를 볼 수 있다.

## Acceptance Criteria
- 피드 카드의 공유 버튼을 누르면 공개 공유 URL이 생성된다
- 공유 URL이 클립보드에 복사되고 토스트로 확인된다
- 공유 페이지에 태그, 카드뉴스 캐러셀, 3줄 요약, CTA 버튼이 표시된다
- 공유 페이지에 공유자/날짜는 표시되지 않는다
- CTA 버튼 클릭 시 랜딩페이지로 이동한다
- OG 메타태그에 3줄 요약과 카드뉴스 1번 이미지가 포함된다
- 유효하지 않은 공유 링크 접속 시 에러 메시지가 표시된다
