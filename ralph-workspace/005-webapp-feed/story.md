# 웹앱 통합 피드

## Description
로그인한 사용자가 웹앱에 접속하면, 자신이 속한 모든 워크스페이스에서 링크가 포함된 메시지들이 최신순으로 피드에 나열된다. 각 메시지 카드에는 공유자, 날짜, 태그, 카드뉴스 캐러셀, 3줄 요약이 표시된다.

## Acceptance Criteria
- 사용자의 모든 워크스페이스에서 완료된(status=complete) 메시지가 최신순으로 나열된다
- 각 카드에 공유자, 날짜, 태그, 카드뉴스 캐러셀(3장 스와이프), 3줄 요약이 표시된다
- 무한스크롤로 추가 메시지를 로딩한다
- 로딩 중 스켈레톤 카드가 표시된다
- 데이터가 없을 때 빈 상태 메시지와 봇 초대 가이드가 표시된다
